# ILMAFOOD

Adalah aplikasi pemesanan makanan online yg di buat dengan laravel, inertia js, react, dan bootstrap.

## FITUR

### Admin

- Dashboard
- CRUD produk
- detail produk
- pesanan
- detail pesanan
- update pesanan
- chat
- laporan pendapatan selama 1 bulan

### User

- pencarian produk
- update profile
- chat
- memesan produk
- kerajang
- login
- register
